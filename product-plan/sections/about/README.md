# About Section

## Overview

A personal section revealing the human behind the code through three themed hobby cards in a matrix layout.

## User Flows

1. **View Section** — See all three interest cards
2. **Quick Jump** — Click jump link to scroll to specific card
3. **Explore Cards** — Hover to see visual effects and details

## Design Decisions

- Single-column layout for focused content
- Three themed cards with distinct data presentations
- Character sheet (TTRPG) with D&D-style stat bars
- Scorecard (Board Games) with game table
- Trail map (Mountain Biking) with difficulty indicators
- Jump links for quick navigation within section

## Components Provided

| Component | Description |
|-----------|-------------|
| `AboutGrid` | Main container with jump links and cards |
| `CharacterSheetCard` | TTRPG-themed card with stats |
| `ScorecardCard` | Board game card with game table |
| `TrailMapCard` | MTB card with trail listings |
