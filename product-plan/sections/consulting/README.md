# Consulting Section

## Overview

Presents three service offerings (Strategic AI, Website Dev, Fractional CTO) with a booking CTA and Local Tech Labs callout.

## User Flows

1. **Browse Services** — View service cards with descriptions
2. **Book Consult** — Click CTA to open contact modal
3. **Visit Local Tech Labs** — Click external link

## Components Provided

| Component | Description |
|-----------|-------------|
| `ConsultingSection` | Main container with intro and cards |
| `ServiceCard` | Individual service offering card |

## Callback Props

| Callback | Description |
|----------|-------------|
| `onBookConsult` | Open contact modal |
| `onLocalTechLabsClick` | Open external site |
